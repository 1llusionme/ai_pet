# Xcode 零基础操作指南（照着点就行）

## 0. 你当前环境

- Xcode 已安装：`16.2`
- 当前还没安装 `xcodegen`，所以走「纯图形界面」方式，最稳。

## 1. 先打开 Xcode

1. 打开 Launchpad。
2. 点击 `Xcode`。
3. 第一次打开如果提示安装附加组件，点 `Install`。

## 2. 新建一个 iOS 工程

1. 在欢迎页点 `Create a new Xcode project`。
2. 选择 `iOS` → `App`，点 `Next`。
3. 按下面填写：
   - Product Name：`MindShadow`
   - Team：先选你的 Apple 账号团队（没有就先留空，后面再配）
   - Organization Identifier：`com.yourname`
   - Interface：`SwiftUI`
   - Language：`Swift`
4. 点 `Next`，保存到任意目录（比如桌面），点 `Create`。

## 3. 把我已经写好的代码拖进去

你只要拖两个目录里的文件：

- `/Users/liyulin/ai_pet/ios/NativeChatScaffold`
- `/Users/liyulin/ai_pet/ios/HybridAppScaffold`

操作步骤：

1. 在 Finder 打开上面两个目录。
2. 在 Xcode 左侧文件树中，选中项目根目录 `MindShadow`。
3. 把这两个目录里的 `.swift` 文件拖到 Xcode 左侧。
4. 弹窗里勾选：
   - `Copy items if needed`
   - `Add to targets: MindShadow`
5. 点 `Finish`。

## 4. 删掉默认入口，换成混合入口

1. 在 Xcode 左侧找到你工程自带的 `MindShadowApp.swift`。
2. 右键删除，选 `Move to Trash`。
3. 确认你已经有这个文件：`MindShadowHybridApp.swift`。

## 5. 改两个地址（非常关键）

打开 `MindShadowHybridApp.swift`，把这两行替换成你的真实地址：

- `apiBaseURL`：后端 API 地址（例如 `https://api.xxx.com`）
- `webBaseURL`：前端 Web 地址（例如 `https://app.xxx.com`）

## 6. 配置真机运行

1. 点击 Xcode 顶部中间的目标设备下拉。
2. 插上你的 iPhone，选择你的设备。
3. 进入 `TARGETS` → `Signing & Capabilities`：
   - 勾选 `Automatically manage signing`
   - `Team` 选择你的 Apple 账号团队
4. 回到顶部，点运行按钮（左上角三角形）。

## 7. 你应该看到什么

- 第一个 Tab 是聊天（原生）。
- 第二个 Tab 是学习资产（WebView）。
- 第三个 Tab 是设置（WebView）。

如果聊天页能发消息、Web 页能打开，这一版就算跑通了。

## 8. 常见报错对照

- 报错 `No such module`：
  - 说明文件没加进 Target，去 File Inspector 勾选 `MindShadow`。
- 报错签名失败：
  - 去 `Signing & Capabilities` 重新选 Team。
- 手机提示不信任开发者：
  - iPhone 设置 → 通用 → VPN 与设备管理 → 信任你的开发者证书。
